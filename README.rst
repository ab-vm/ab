=====
Ab VM
=====

|Travis Build Status| |Appveyor Build Status|

A WebAssembly Virtual Machine. Ab documentation is hosted at `Read the Docs`_.

Major Components
================

- Ab: Virtual Machine
- Abi: Ab Interactive
- Om: GC Object Model
- Pith: Core C++ Utilities

=============================

Ab is built on `Eclipse OMR`_
